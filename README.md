# Formative Assignment: Linear Equation's Quiz
A coding tool that gives you five questions linear equation questions to answer. The tool will tell you whether you are right or wrong and give you a score out of five at the end.

## User Documentation

**Step 1: Run the code**
Press the play icon in the top right corner to run the code in the terminal.



**Step 2: Answer Question 1**
When prompted, answer the first question. If you are incorrect, the code will prompt you to try again. You get an unlimited number of attempts until you answer correctly.

**Step 3: Answer Every Question**
Continue answering the next 4 questions. Once all have been answered correctly, the quiz will finish, letting you know your score out of 5.

**Step 4: Replay Anytime**
To replay, type "clear" in your terminal and then press the play icon to run the quiz again.
